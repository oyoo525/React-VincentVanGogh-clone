# Vincent Van Gogh Museum
> ### museum 사이트 클론코딩
> 1인 프로젝트 <br>
> 2024.02.28 ~ 진행중 <br>
<br>


## 사용기술
> React, React-Router-DOM, Typescript, CSS, bootstrap-icon
<br>

## 개발환경
> - Windows 10
> - VS Code
<br>

## 프로젝트 소개
> ### 프로젝트 목적
> - 프론트단에 대한 숙련도 향상을 위한 프로젝트
> - 최근 많이 사용하는 React 프로그램 활용
> ### 프로젝트 구현내용
>
> ### 프로젝트를 통해 배운점
> - flex를 사용하면 width 안으로 채워지므로 추가 설정이 필요하다는 것을 알게 되었다.
> -
